## Why this matters

AI systems can generate recommendations, prepare decisions, route work, operate tools, and accelerate workflows.

But faster AI-supported work creates a governance question:

```text
Who is responsible when AI-supported output becomes action?
```

RABA explores how organizations can make that responsibility visible before, during, and after AI-assisted decisions.

- canon

Important boundary:

```text

Multi-AI agreement is not Human Owner approval.
External signal is not validation.
Repository presence is not canon.
Technical trace is not enough without responsibility trace.
```
